最后一个坑

本地测试正常，npm发布后再npm install -g安装执行就报错

着实纠结好久，最后试着是在入口文件中加
#!/usr/bin/env node

虽然可以了，但是原理不太理解，#!/usr/bin/env node是Linux系统下，在shell脚本中，通过#!注释来指定当前脚本使用的解析器。
